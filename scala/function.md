### 함수 리터럴

- 파라미터 이름의 목록이 괄호 안에 오고, 그 뒤에 오른쪽 화살표, 마지막으로 함수의 본문이 있어야한다.
```scala
(x: Int, y: Int) => { x + y } //함수 본문이 한 줄이라면 중괄호 생략 가능.
```

### Syntax Sugar

- 함수의 인자가 하나일 경우 다음과 같이 소괄호 대신 중괄호로 묶을 수 있다.
```scala
println("hello scala")
println { "hello scala" }
```
> 단 두 개의 인자를 전달하라고 하면 컴파일 에러가 발생한다.
